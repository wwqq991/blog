简单总结一下我发现的各种蚊子腿（申请条件就不特意写了，没写就是无门槛或者低门槛）。

有时候也是好奇社区打野的都是哪打来的。

# 1. Blaze

偶然发现的，模型变动且不稳定。

- 网站：<https://blazeai.boxu.dev/>
- 申请需求：邮箱 + Discord
- 额度：1000 次，不同模型倍率不同
- 模型：不稳定，经常变，目前是 grok-4.20-fast、kimi k2.6、glm5.1 这些
- 速度：只能说勉强能用
- URL：`https://blazeai.boxu.dev/api`

# 2. 硅基流动

老生常谈，注册实名给代金券，不多说了。

# 3. Groq

这位更是没法用。

- 网站：[Groq](https://groq.com/)（Groq is fast, low cost inference.）
- 申请需求：忘了，反正很简单
- 额度：没印象了
- 模型：Llama 系列，Qwen 系列（其实就一个）。Groq 也许有用，但是 Groq 有用不太可能

# 4. 各种中转站

各种中转站的签到奖励，过于蚊子腿，不说了。

# 5. GitHub 公益项目

GitHub 上的一个项目，有免费模型。日常用还可以，RP 就没法用了。不多说。

# 6. OpenRouter

老生常谈了。

- 网站：[OpenRouter](https://openrouter.ai/)
- 额度：1000 次（不过至少要充值 10 美元）
- 模型：理论上都有，既然是白嫖，那就搜一下 free 就好。现在有 kimi k2.6（不过大概是要排队），还有一些模型可以拿去填表（大概吧）

# 7. OpenCode

不定期的免费模型。

- 网站：[OpenCode](https://opencode.ai/)（开源 AI 编程代理）
- 额度：免费差不多就是无限？
- 模型：这就得看 OpenCode 的心情了，现在是有 DSv4Flash 和 Qwen、MiMo 这种

# 8. Navy

- 网站：[NavyAI](https://navy.ai/)（Unified AI API）
- 额度：每天 15w
- 条件：Discord 登录
- 模型：不少，免费用户 160 个模型可用（别被骗了，其实很多模型都是凑数的），付费 184 个模型。国模（DS、GLM 之类的）、Gemini、GPT，不同模型倍率不同（也许能搞好几个 Discord 账号一起用？）。Claude 没有，或者说免费用户没有

# 9. Fyra

- 网站：[Fyra.im](https://fyra.im/)（统一的 LLM API）
- 额度：每天 100w
- 模型：怎么全是 Qwen，我记得以前免费账号还有 DeepSeek 来着

# 10. NVIDIA NIM

老生常谈。

- 网站：[NVIDIA NIM](https://build.nvidia.com/)（Try NVIDIA NIM APIs）
- 额度：理论上不限量
- 注册需求：一个邮箱（可以是域名邮箱），一个手机号（验证之后才有权限，可以 +86），也是来回白嫖了 4 个 key

# 11. FreeModel

到底怎么破开 GPT...

- 网站：[FreeModel AI](https://freemodel.ai/)
- 额度：我记得好像是 300 美元？
- 注册：邮箱 + 86 手机号或 Telegram 号（其实就是所有手机号都行）
- 模型：全是 GPT
- 不玩 RP 拿来接 CC 也还可以

# 12. 商汤日日新

白给的 DeepSeek，不过好像很难破开，最后拿去接 Hermes 了。

- 网站：[日日新](https://platform.sensenova.cn/)（商汤科技大模型平台）
- 额度：每模型 1,500 次调用 / 5 小时
- 模型：DSv4Flash、SenseNova 6.7 Flash-Lite、SenseNova U1 Fast

# 13. Commonstack

这个已经有人说过了，就不细说了，依旧闲得无聊古法注册 121 个号。

# 14. GeneralCompute

这个也有人说过了，也不细说，只有 DS3.2 和 MiniMax，感觉用处不大。

# 15. Kiro

Kiro 能白给一个月 Pro，不过我的卡被拒了，就离谱，懒得弄了。

# 16. Bluesminds

白给 100 额度，和一个月的 vip 分组，不确定能不能批量注册

- 网站：[Bluesminds](https://api.bluesminds.com/)
- 额度：100 额度
- 模型：主流都有，无 claude

vip 分组需联系管理员……不开 vip 渠道部分模型被限制了，具体还没测试。不知道是临时调整还是以后都这样

# 17. verdent

免费试用七天

- 网站：[Verdent AI 定价 | 订阅计划](https://www.verdent.ai/zh-CN/credits-detail)
- 额度：100 积分
- 模型：Claude Opus 4.8 / Sonnet 4.6、GPT-5.5、Gemini 3.1 Pro、GLM-5.1、Kimi K2.6

只允许在官方软件里或者作为 vscode 插件使用

实测 edu.kg 域名邮箱注册成功，同时支持 GitHub 和 gmail 注册，各位自行发挥吧
