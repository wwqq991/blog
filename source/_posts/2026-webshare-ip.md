# 1. 前言

前段时间白嫖了 Gemini 的一年 pro 会员，为了避免封号，我开始 IP 焦虑，总觉得自己的 IP 不够好，于是乎四处寻找静态住宅 IP，也是踩了不少坑，最后找到了 webshare。

# 2. webshare

链接：<https://www.webshare.io/?referral_code=c93yo4drl2hk>

一个静态住宅 IP，一个月 1 美元，一年差不多 8 美元。

有点不想写了。

买的时候，IP 数量记得选“习俗”，我也不知道为啥自动翻译给翻译成习俗了。

# 3. 购买

需要一张卡来付款，正经信用卡或者 bybit 卡也可以。

没卡的可以来：<https://www.bybitglobal.com/invite?ref=BDGMXJD>

申请一个，反正也是免费开卡。

就这样吧，懒得写了。

# 4. 问题

不对，还得说一下，这个 IP 用 GPT、Gemini、TG 都行，但是别拿来用 Claude。因为我的号已经被封了，就难绷。

另外，不适合拿来做跨境电商，我看已经有几个平台把这个 IP 拉黑了。真想用的话要筛选 IP 段，有点麻烦，我懒得搞。而且这大部分是单 ISP，某些是双 ISP，抽奖这一块。
